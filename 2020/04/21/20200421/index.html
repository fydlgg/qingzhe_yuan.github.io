Y原创 | 黑科技：Rhino中3秒快速生成剖面并且与模型同步更新

发表于 2020-04-21 | 分类于 Y原创 |

写在前面

最近小编在绘制剖面的时候，觉得目前的用一个平面Surface去跟整个建筑进行intersect的工作流不是很有效率，一方面是因为相交之后的线型并不会保留图层的信息，另一方面是每一次更新都要进行一系列步骤：相交—>水平旋转至X轴—>竖向旋转至水平—>导出。希望找到一个更有效率的工作流，可以更加快速的预览剖面以及导出。通过一系列搜索之后，发现了SectionTool这一个非常好用的剖面以及平面生成插件工具。

插件优点


短短几秒，就已经生成了一个储存着剖面信息的模型了（生成了一个名为SEC_00的剖断线）。有多种方式可以预览剖面：第一种可以剖切模型看到剖面：


另外还有一种方式可以预览：


这个插件最厉害的地方是可以同步更新：


当我将地面移动后，可以一键更新剖面预览。

在这种快速更新的功能下，可以延伸出更多的可能。可以进行模型空间的预览，甚至可以在rhino中进行初步的平剖面的绘制。如何在rhino中进行高效的CAD绘制可以参考一下链接：走捷径 | 论如何科学地用Rhino画CAD。

另外，这个插件还可以高效地绘制各种各样的剖面，可以参考蒯神的教程：Rhino画剖面的各种姿势~

不过这些教程还没有涉及到这个插件非常有用的功能，实时更新剖面（平面也是一种剖面）。本期就跟大家介绍这个功能的使用方法。

插件获取方式

第一种方式是通过Rhino插件的官网，链接如下：https://www.food4rhino.com/。 其中搜索Section Tool，第一个结果就是。


注意下，两个文件都要下载，第一个是安装包，另一个是案例及使用说明。如果是Rhino5版本可以下载第三个rhino5兼容的安装包。


第二种方式是通过Section Tool插件的官方页面：https://wiki.mcneel.com/labs/sectiontools。


点击下载后需要输入邮箱，随便输入一个就行，因为并不会有任何的验证码，就可以直接下载。

最后一种办法，我会在文章末尾提供百度网盘链接供下载。

插件安装

下载之后会有两个文件：


.rhi文件可以直接双击安装，.zip文件中是案例模型和说明书。说明书中有详细的使用说明，虽然是针对于上一版Rhino5的，但是基本差别不大。遇到版本间的区别，我会跟大家说明的。接下里我就按照说明书的顺序介绍插件的使用方式。

使用手册

Toolbars and Menu

调出SectionTool的工具栏的方法：文件栏中的Tools —>Toolbar Layout。


在打开的菜单中勾选Section Tool即可在rhino界面中显示SectionTool的工具栏。


stCreate Options（One by one & Array of）


第一个命令是创建一个或者一系列的剖面。快捷键分别是stCreate和stCreateArray。每次调用这个命令时，会有很多的Options需要进行设置：


我们先用默认的设置创建一个剖面，根据结果跟大家介绍每一个设置的含义。


设置1：Dir。

这个设置有三个选项：Horizontal、Vertical、Custom，功能是控制剖面线在top视角下水平、垂直或者自定义方向。演示如下：


设置2：ExtrudeDir。

这个设置有两个选项：CplaneNormal、Custom。这个设置是控制剖面线的剖切方向，这个设置可以实现平面的剖切。


从剖面箭头的方向就可以看出来，新创建的剖面的剖切方向是垂直向下，也就是平面的剖切方向。

设置3：Replace。

这个设置是个布尔值（Yes、No），功能是替换相同名字的剖面。但是好像由于设置了命名自动编号的功能，这个Replace就没用了。当命名同一名字时，系统会自动加尾缀：“01”、“02”，导致不会出现重名现象：


设置4：Name。

很好理解，就是对于剖面进行命名。不过系统会自动增加后缀编号，从“00”开始。

设置5：LoadFromFile。

这个设置是可以从外部导入剖面。而具体的格式，使用说明中也有详细的描述：


文件格式可以是.txt, .dat, .csv。包含的数据只是剖面名字和相对于基点（BasePoint）的位置。这个功能实用性不强，因为可以定义的只有位置，而且是相对于基点的一维位置。但是非常适合用于设置平面剖切面的位置。

示例模型的高度为20feet，希望平面的剖切位置设置在5feet的位置（单位我就不改了）。在txt文件中输入剖面名称和位置：


为什么是-5？因为平面的剖切方向是竖直向下，而剖切面是楼板上5feet，所以输入-5。


这样就可以精确的得到平面的剖切位置，不用先设置，再移动了。

设置6：Attributes。

这个设置有两个选项：ByObject、ByDisplay。剖切生成的物件的图层是默认按照Curve、Hatch和Solid区分好了，只有颜色属性可以调整。ByDisplay是生成的物件的颜色根据物件的显示颜色；ByObject是根据物件的自身属性。

ByDisplay：


生成物体的颜色根据显示的颜色。虽然DisplayColor的值为By Layer，但是显示为什么色就是什么色。

ByObject：


生成物体的颜色根据原来物体的属性。原物体的DisplayColor的值为By Layer，所以新生成的物体的值为By Layer。新生成的图层默认颜色为黑色，物件自然默认为黑色。

设置7：Solid Mode。

这个设置有三个选项：Suface、Hatch、Both。分别对应着三种显示模式：显示面、显示Hatch、都显示。


从左到右分别是：Both、Surface和Hatch。

设置8：HPattern。

这个设置可以调整填充的图案。不过这个设置有个bug，就是只能识别在模型内使用过的Hatch图形，所以使用前最好先填充一次。另外，Rhino是可以导入天正和cad中的图案的，网上有很多教程，可以搜索下。


设置9&10：HRotation和HScale。

这个设置调整填充图案的角度和大小，就不详细介绍了。

设置11：Labels。

这个是设置设置剖面标识类型的，有四个设置选项：ShowAll、CurveOnly、DotOnly、HideAll。


这就是四种显示类型，可以自己尝试下。

设置12：ArrowScale。

设置剖切符号的大小。


至此，已经介绍了生成一个剖面所涉及到的所有设置，接下来介绍其他按钮命令的功能。

Edit Section


这个按钮是用来修改某个剖面的设置的，快捷键是：stEditSections。（第二个命令是用来更新剖面，之后会讲）。


其中有一个设置是生成剖面时没有的——Flip。用来改变剖面的方向，其他设置上述说明中都介绍过。

Edit Section Object / Hint


这个按钮左键用来调整需要剖切的物体，右键用来调整剖切符号的格式。快捷键分别是：stEditSectionObject/stEditSectionHint。

Move Section


不用多说，用来移动剖切面。快捷键是：stMoveSection。


Create 2D Layout / Static Nested Layout


左键是创建一个可以随时更新的2D剖面，右键时创建一个静态的剖面（不会随着模型或者剖面移动产生改变）。


左侧时2D Layout，右侧是静态的剖面。
