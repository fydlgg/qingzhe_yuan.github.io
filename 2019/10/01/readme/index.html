写在前面

  工作之后计划学习grasshopper，这已经是第n次决定去深入学习，但是之前急功近利，浅尝辄止。所以这次决定边学边记，建立一个公众号进行记录整理，整理自己的思路，让学习更加成为体系。从2019.10.01的假期开始，到现在2020.02.10已经更新三个月，也更新了一些文章，正值Noah的发布，目前包括两个方面：1.grasshopper自学笔记；2.Noah电池详解。

  但是在之后的自我复习，以及平时对于已写文章的使用时，发现手机端，由于屏幕的限制，对于代码和电池的显示很受限制。另外之后我将逐步开始RhinoPython的学习，对于代码学习复习来说，有时Ctrl+C和Ctrl+V还是很重要的。所以建立了这个网站同步更新相关学习笔记。这个文档是基于Markdown进行编辑，对于代码和图标展示非常的方便，如下便是对于代码的展示，之后也将以这种形式与大家见面：
