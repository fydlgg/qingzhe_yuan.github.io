Grasshopper背景

Grasshopper是由Robert McNeel＆Associates的David Rutten开发的可视化程序编辑器。作为Rhino的插件，Grasshopper与功能强大的建模环境集成在一起，供建筑，工程，产品设计等多个领域的创意专业人员使用。同时，Grasshopper和Rhino为我们提供了定义模型的精确参数控制的机会，探索生成的设计工作流的能力以及开发高级编程逻辑的平台，而所有这些都在直观的图形界面中进行。

Grasshopper的起源可以追溯到Rhino版本4中的“Record History”按钮。借助此内置功能，用户可以随时随地在后台存储建模过程。例如记录了四个曲线的放样过程，然后编辑了其中一条曲线的控制点，则曲面几何形状将更新。早在2008年，David提出了一个问题：“是否对建模过程进行更明确的控制？”，Grasshopper便因此诞生了。这样就可以对建模历史进行详细的编辑，并使用户能够开发超出Rhino内置现有功能的功能逻辑序列。六年后，Grasshopper现在是一个强大的可视化程序编辑器，可以通过外部开发的组件进行扩展。

本入门课程侧重于基础知识，提供研究Grasshopper的常规操作以及所需要的核心知识以，以帮助您在自己的创造性实践中进一步使用。在准备继续学习下文之前，让我们了解下什么是视觉编程、Grasshopper界面、基础术语、视口反馈和用户体验等特性。

可视化编程是计算机编程的一种规范，其中，用户以图形方式而不是文本方式操作逻辑元素。一些最著名的文本编程语言，例如C＃，Visual Basic，Processing ，以及更接近Rhino的语言– Python和Rhinoscript要求我们编写受特定语言语法约束的代码。相比之下，可视化编程让我们可以将功能块快速连接，进行一系列操作，其中唯一需要的“语法”是电池块输入或接收适当类型的数据。在理想情况下，我们可以根据需求进行组织。详情请参阅“1.5 数据流匹配和使用数据树设计”部分。视觉编程的这一特征避免了尝试学习一种新语言（甚至是口头语言）时通常会遇到的障碍。


要使用Grasshopper及其可视化编程功能，我们需要从Grasshopper3D.com网站下载并安装该程序。安装完成后，我们可以在Rhino命令行输入“ Grasshopper”来打开插件。我们将看到Grasshopper加载提示，然后是Grasshopper编辑器窗口。现在，我们可以将电池块添加到“画布”，将它们用“电线”连接，然后将文件保存为.ghx格式。


我们可以在Grasshopper画布中创建“Slider”电池以控制几何形状，也可以将在此输入的数据与Rhino视口中的预览连接起来。这种连接是实时的。如果我们调整Silder的数据，Rhino窗口中的物件会更改。为了Grasshopper的使用方便，我们看到Rhino窗口中的几何预览是轻量级显示，并且会自动更新。请注意这种同步的关系，因为当您的定义变得更加复杂时，巧妙地管理了数据流，注意坚持的状态以及Rhino视窗中的模型可以高效的避免一些不必要的麻烦。

1.1.2.1 WINDOWS标题栏

GH编辑器窗口标题栏的操作与Microsoft Windows中的大多数其他的标题栏不同。如果未最小化或最大化窗口，则双击标题栏会将对话框折叠到屏幕上的最小化栏中。这是在插件和Rhino之间切换的好方法，因为它可以最小化编辑器，而无需将其移至屏幕底部或其他窗口后面。


请注意，如果关闭编辑器，Rhino视口中的Grasshopper预览将消失，但实际上不会关闭该文件。下次在Rhino对话框中运行“ Grasshopper”命令时，该窗口将以相同的状态返回，并加载相同的文件。这是因为一旦从命令提示符启动它，您的Grasshopper窗口将保持激活状态，直到该Rhino实例关闭。

1.1.2.2 主菜单栏

标题栏类似于经典的Windows菜单，除了右侧的文件浏览器控件（部件3，请参阅下一节）。“File”菜单提供了一些常用功能（例如，“新文件”，“打开”，“保存”等），以及一些实用工具，可用于导出当前Grasshopper文档的图像（请参阅快速导出图像和导出高分辨率图像的说明）。您可以使用“View”和“Display”菜单来控制用户界面的不同方面，而“Solution”菜单可让您管理Grasshopper处理数据的不同属性。

值得注意的是，可以通过“File”菜单下的“Preference”对话框来控制许多应用程序设置。其中，“Author”部分可让您设置个人相关，该数据将与每个Grasshopper文档一起存储，而“Display”部分，可让您对界面的外观进行精细控制。使用“FIle”部分，您可以指定自动保存文件的频率和存储位置（以防应用程序意外关闭或崩溃）。最后，“Solution”部分使您可以管理可以扩展功能的核心和第三方插件。


1.1.2.3 文件浏览器控制

通过文件浏览器，您可以通过此下拉列表选择不同的加载文件，从而在不同的加载文件之间快速切换。通过“文件浏览器”下拉列表打开文件，使您能够快速复制和粘贴不同文件中的项目。只需在“浏览器控制窗口”中单击活动文件名，就会显示所有打开文件的列表（以及每个打开的定义的小缩略图），以便于访问。您也可以按Alt + Tab快速在任何打开的Grasshopper文档之间切换。

当然，您也可以通过标准的“Open File”对话框来加载任何Grasshopper文档，尽管您也可以将任何Grasshopper文件拖放到画布上以加载。

Grasshopper是一个可以在Rhino的“顶部”运行的插件，因此具有自己的文件类型。默认文件类型是二进制数据文件，扩展名为.gh。另一种文件类型称为Grasshopper XML文件，其扩展名为.ghx。XML（可扩展标记语言）文件类型使用标签定义对象和对象属性（非常类似于.HTML文档），但使用自定义标签定义对象和每个对象中的数据。因为XML文件被格式化为文本文档，所以您可以在诸如NotePad之类的文本编辑器中打开任何Grasshopper XML文件，以查看幕后进行的编码。

Grasshopper有几种不同的方法可以用来打开文件，使用这种方法时，您需要指定要使用的选项。

  • 打开文件（Open）：顾名思义，此文件选项将仅打开您拖放到画布上。

  • 插入文件（Insert）：您可以使用此选项将现有文件作为分离的组件插入当前文档中。

  • 分组文件（Group）：此方法会将文件插入现有文档，但是将所有对象分组在一起。

  • 群集文件（Cluster）：类似于组功能，此方法会将文件插入现有文档中，但将为整个对象组创建一个Cluster对象。

  • 检查文件（Examine）：允许您以锁定状态打开文件，这意味着您可以浏览特定文件，但不能对定义进行任何更改。

Grasshopper还具有自动保存功能，该功能将根据特定的用户设定定期触发。在主菜单栏上的“File”菜单下可以找到自动保存首选项的列表。当Rhino文件关闭时，将出现一个弹出对话框，询问您是否要保存Rhino关闭时打开的任何Grasshopper文件。


1.1.2.4 组件（电池）面板

该区域将组件（电池）分为类别和子类别，类别显示为不同的选项卡，子类别显示为下拉菜单。所有组件都已经分类为某个类别。这些类别按照清晰的逻辑区分，快速找到所需的特定组件（例如，所有原始数据类型的“ Params”或所有曲线相关工具的“ Curves”）。要将组件（电池）添加到画布，可以单击下拉菜单中的对象，也可以将组件直接从菜单拖动到画布上。


由于每个子类别中的组件数量可能超出调色板所能容纳的数量，因此每个面板上显示的图标数量有限。可以调整组件面板的高度和Grasshopper窗口的宽度，以在每个子类别中显示更多或更少的组件。要查看给定子类别中所有组件的菜单，只需单击每个子类别面板底部的黑条。这将打开一个下拉菜单，通过该菜单可以访问该子类别中的所有组件。

1.1.2.9 查找功能

实际上有数百种Grasshopper组件可供您使用，作为初学者，想要在组件面板中查找特定组件会比较困难。快速方法是双击画布上的任意位置以启动针对您要查找的组件的搜索查询。但是，如果我们需要找到已经放置在画布上的特定组件该怎么办？不用担心。通过右键单击画布上的任意位置或按F3键，可以调用Find功能。首先输入要查找的组件的名称。

查找功能使用了非常复杂的算法，这些算法不仅可以在画布内搜索组件名称实例（组件名称是在组件面板下找到的组件的标题，我们作为用户无法更改，但我们可以为组件指定唯一签名（称为昵称）。Find功能还可以在画布上搜索任何组件类型，或对于文本面板，涂鸦和组内容中搜索。查找功能找到匹配项后，它将自动使其余定义变灰，并在突出显示的组件周围绘制虚线。

对于Grasshopper的分析

Grasshopper允许您创建可视化的程序。这些文件由通过导线连接的节点（电池）组成。这一章介绍Grasshopper电池以及如何与它们交互，以开始构建grasshopper文件。

Grasshopper的对象类型

grasshopper包括两种主要的用户对象类型：参数和组件。参数存储数据，而组件执行产生数据的动作。理解Grasshopper的最基本方法是：我们将使用数据来影响组件所产生的动作。

1.2.1.1 参数 Parameters

参数存储在文件中的数据，数字，颜色，几何图形等。参数是容器对象，通常显示为带有单个输入和单个输出的小矩形框。我们可以通过图标的形状，知道哪些是参数，因为所有参数对象的图标周围都有一个六边形边框。

几何参数可以从Rhino引用几何，也可以从其他组件继承几何。点和曲线对象都是几何参数。


“输入参数（input param）”是动态接口对象，可让您与文件中物体进行交互。数字滑块和图形映射器都是“输入参数”。


1.2.1.2 组件 Component

组件根据它们收到的参数执行操作。有许多类型的组件可用于不同的任务。

1.乘法部分是一个运算符，用于计算两个数的乘积
2.“除”组件对几何进行操作，将曲线分为相等的段
3.Circle CNR组件根据输入数据构造一个圆的几何形状。中心点，法线向量和半径。
4.放样组件通过放样曲线构造曲面。

1.2.1.3 组件颜色

我们可以根据它们的颜色得到一些有关每个对象状态的信息。让我们看一下Grasshopper的默认颜色编码系统。

包含警告的参数显示为橙色框。任何无法收集数据的组件在Grasshopper文件中均被视为可疑对象，因为它对整个gh文件没有任何帮助。因此，所有组件（刚添加时）均为橙色，表示它们不包含任何数据。默认情况下，橙色的参数和组件在右上角也有一个小气球。如果将鼠标悬停在此气球上，它将显示有关组件为何向您发出警告的信息。一旦输入对应的数据，它将变为灰色，并且气球将消失。

数据类型

大多数参数可以存储两种不同类型的数据：临时数据和永久数据。临时数据从一个或多个来源继承，并且每当有新的输入时便会被替换（即重新收集）。永久数据是由用户专门设置的数据。

1.2.3.1 永久性数据

永久数据可通过Param接收器进行设置，并且根据参数的类型具有不同的接收器。例如，“Point Param“允许您通过其菜单设置一个或多个点。

将Point Param从组件面板拖放到画布上时，该参数为橙色，表示生成警告。警告只是通知您该参数为空（它不包含任何永久数据，并且未能接受临时数据），因此对整个计算的结果没有影响。参数的弹出菜单提供了两种设置永久性数据的方法：单个和多个。右键单击参数以设置多个点。单击这些菜单项之一后，Grasshopper窗口将消失，并且将要求您在Rhino视窗中选择一个点。


一旦设置了点数据后，可以按Enter键，它们将成为永久数据的一部分。这意味着该参数现在不再为空，并且从橙色变为灰色。（请注意，由于没有更多警告，因此右上角的信息提示框也消失了）。此时，您可以将此参数中存储的点用于任何后续的计算。

点、平面和向量

一切的物体都从点开始。点仅是一个或多个称为坐标值得组合。坐标值的数量与它所驻留的空间的维数相对应。点，平面和向量是在Grasshopper中创建和转换几何的基础。


1.3.1.1 点

三维空间中的点具有三个坐标，通常称为[x，y，z]。二维空间中的点只有两个坐标，称为[x，y]或[u，v]。二维空间绑定到有限的曲面中，它仍然是连续的，也就是说，假设曲面上有无数个点，但是这些点之间的最大距离是有限的。二维空间坐标只有在不超过面域范围区间的情况下才有效。在示例下图中，对于[u]和[v]方向，范围都设置为0.0到1.0，但是它可以是任意的有限域。而坐标为[1.5，0.6]的点将在曲面外部某处，因此无效。

由于某个特定区间的曲面是真实位于三维世界空间中的，因此我们可以将曲面坐标转换为三维世界坐标。例如，表面上的点[0.2，0.5]与世界坐标中的点[1.8，2.0，4.1]相同。一旦使表面变形或变形，与[0.2，0.5]对应的三维坐标将改变。


这是一个很难理解的概念，但是联想到您在地球以及太空中的位置关系，则有助于理解。我们倾向于使用局部坐标系来描述我们的位置：“我坐在电影院第七排的第三位”，“我坐在后排座位”。如果您所在的汽车在行驶中，即使您保持在相同的后座“坐标”中，您在世界坐标中的位置始终在变化。

1.3.1.2 向量

向量是描述方向和大小的几何量。向量是抽象的；即。它们代表方向和数量，而不是单纯的几何元素。

向量与点是无法区分的。也就是说，它们都是由三个数字组成的列表，因此没有办法确定某个列表是表示点还是向量。但是两个定义是有实际的区别。点是绝对的，向量是相对的。当我们将三个双精度数的列表视为一个点时，它表示空间中的某个坐标，当我们将其视为矢量时，它表示某个方向。向量是空间中的箭头，总是从世界原点（0.0，0.0，0.0）开始，在指定坐标处结束。


1.3.1.3 平面

平面是“平”的，并且在两个方向上无限延伸，从而定义了该部分的坐标系。在Rhino中，平面不是真正的对象，它们用于定义三维世界空间中的坐标系。实际上，最好将平面视为一种特殊的向量组合，它们都是只是数学构造，不是真实的物体。

1.3.3.3 条件判断

几乎每种编程语言都有一种条件语句进行判断的方法。程序员经常会创建一段代码来询问一个简单的的问题。如果楼层轮廓的面积超过了要求，该怎么办？或者，如果我屋顶的弯曲程度超过实际值怎么办？这些代表着高层次的抽象思想。计算机程序具有分析问题并根据该问题的答案执行措施的能力。

让我们看一个程序如何解释的非常简单的条件语句：如果对象是曲线，则将其删除。这段代码首先查看一个对象，并确定一个布尔值，确定它是否为曲线。没有回旋的余地。如果对象是曲线，则布尔值为True，反之False（如果对象不是曲线）。语句的第二部分根据条件语句的结果执行操作。在这种情况下，如果对象是曲线，则将其删除。此条件语句称为If语句。在Operators类别下有四个条件运算符，它们可以判断条件并返回布尔值。

Similarity近似判断组件也是对于两个数据列表进行判断，并测试两个数字之间的相似性。它与“相等Equality”组件比较两个列表的方式几乎相同，但有一个例外：它有一个百分比的输入端，用于定义列表A和列表B判断是否相等的差值范围。相似性组件还具有一个输出，该输出确定两个输入列表之间的差值的绝对值。


大于Larger Than组件也是输入两个数据列表，并确定列表A的第一项是否大于列表B的第一项。这两个输出使您可以确定是否要根据大于（>）或大于等于（> =）条件来评估两个列表。


小于Smaller组件执行与“大于”组件相反的操作。“小于”组件确定列表A是否小于列表B并返回布尔值列表。同样，这两个输出使您可以确定是否要根据大于（>）或大于等于（> =）条件来评估两个列表。

1.3.5.1 布尔值

数字变量可以存储整个范围的不同数字。布尔变量只能存储两个值，分别称为Yes或No，True或False，1或0。显然，由于它们的范围有限，我们从不使用布尔值执行计算。我们使用布尔值来判断条件。


在Grasshopper中，布尔值可以多种方式使用。布尔参数组件是一个或多个布尔值的容器，而布尔切换使您可以在单个true和false值之间快速切换作为输入。


Grasshopper还具有测试条件并输出布尔值的功能。例如，通过Includes组件，您可以测试数字值以查看其是否包含在区间中。

  3. Or（或门）

1.4.4.1 Nurbs Curve

Degree：Degree是一个正整数。该数字通常是1、2、3或5，但也可以是任何正整数。曲线的Degree决定了控制点对曲线的影响范围。程度越高，范围越大。NURBS直线和Polyline通常是1级，NURBS圆（弧）是2级，大多数自由形式的曲线是3或5级。

控制点Control Point：控制点是至少度数+1点的列表（例如，直线是1Degree，至少有起止两个点）。更改NURBS曲线形状的最简单方法之一就是移动其控制点。

权重Weight：控制点有一个属性，称为权重Weight。权重通常是正数。当曲线的控制点都具有相同的权重（通常为1）时，该曲线称为非有理值，否则称为有理曲线。大多数NURBS曲线都是非理性的。其中一部分NURBS曲线（例如圆形和椭圆形）总是有理的。

结点Knots：结点Knots是（Degree + N - 1）个数据的列表，其中N是控制点的数量。

编辑点Edit Points: 曲线上的编辑点根据结点Knots分布。编辑点就像控制点一样，只不过它们始终位于曲线上，移动一个编辑点通常会更改整个曲线的形状（移动一个控制点只会局部更改曲线的形状）。当您需要曲线内部的一个点精确通过特定位置时，编辑点很有用（参考GH中Interpolate电池）。

列表 What is List

从数据流的角度学习Grasshopper很有帮助，因为图形界面目的就在于使数据输入和输出特定类型的数据。而列表正是这些数据的重要组合方式。所以，了解如何操作列表数据对于了解Grasshopper至关重要。

Grasshopper具有两种类型的数据：永久性数据和暂时性数据。无论数据类型，Grasshopper通常也会将此数据存储在数组（列表）中。

将数据存储在列表中时，了解列表中每一项（Item）的位置会很有帮助，这样我们就可以开始访问或操作某些项。项在列表中的位置称为其索引号（Indices）。


  1. 列表
  2. 索引号

列表的第一个索引号始终为0，不是1。因此，当我们谈论列表的第一项时，实际上是指与索引号0相对应的项。

例如，如果我们要计算右手的手指数，则很有可能您的手指数是1到5。但是，如果此列表已存储在数组中，则我们的列表应该从0到4。请注意，列表中还有5个项目。只是该数组使用基于0的计数系统。列表中存储的项并不仅仅是数字。它们可以是Grasshopper支持的任何数据类型，例如点，曲线，曲面，网格等。

通常，查看列表中存储的数据类型最简单的方法是将Panel（Params/Input/Panel）连接到某个组件的输出端。默认情况下，Panel会在面板左侧显示所有的索引号Indices，并在右侧显示数据项Item。可以通过右键单击“Panel”并单击子菜单中的“Draw Indices”项来打开和关闭索引号。

Nurbs 曲面

NURBS曲面（non-uniform rational B-splines）是一种将曲面数学化的表示形式，可以精确地建模任何形状。从简单的2D线，圆，弧或方体到最复杂的3D自由形有机表面或实体。由于其灵活性和准确性，NURBS模型可用于从平面设计，动画建模到工程制造中的任何过程。

除了一些原始的曲面类型（例如球体，圆锥，平面和圆柱体），Rhino还支持三种自由曲面类型，其中最有用的是NURBS曲面。与曲线类似，所有可能的表面形状都可以用NURBS曲面表示。到目前为止，它也是最有用的曲面定义方式，也是我们将重点学习的。

1.5.1.1 Nurbs 曲面

NURBS曲面与NURBS曲线非常相似。使用相同的算法来计算形状，法线，切线，曲率等属性，但是存在一些明显的差异。例如，曲线具有切线向量和法线平面，而曲面具有法线向量和切线平面，这意味着曲线缺少法向量而曲面没有切向量。在NURBS曲面的情况下，实际上存在两个方向，因为NURBS曲面是{u}和{v}曲线的矩形网格。即使这些方向通常是任意的，我们还是最终还是使用它们，因为它们使我们的更加方便地对于曲面进行分析和细化。


您可以将NURBS曲面视为沿两个方向延伸的NURBS曲线的网格，由多个控制点以及该曲面在u和v方向上的度数（Degree）定义。更高的Degree表示NURBS曲面可以高效地存储和表示高度精确的自由曲面。

Grasshopper处理Nurbs曲面的方式与Rhino相似，因为应用了同一种算法。但是，Grasshopper为了保证运算的效率，有些曲面的显示略有不同，显示质量较低，以高效的运算。您可能会注意到GH中的曲面中有一些不顺滑的面，但这是意料之中的，并且仅是Grasshopper显示设置的结果。任何Bake的几何体仍将使用较高的显示设置。

1.5.1.1 曲面投影

在上一节中，我们解释了Nurbs曲面包含由u和v域定义某点的坐标空间。这意味着可以将由x和y坐标定义的二维几何图形映射到曲面的uv空间。几何形状将相应根据曲面的曲率而拉伸和变化。这与将二维几何体简单地投影到曲面上不同，将从从二维几何体绘制向量，直到它们与曲面相交为止。


您可以将投影视为将阴影投射到曲面上的几何图形，将映射视为在曲面上拉伸几何图形。

  1. uv坐标定义的映射几何
  2. 将几何体投影到表面上

正如二维几何图形可以投影到曲面的uv空间上一样，盒子包含的三维几何图形也可以映射到表面上进行相应的扭曲。此操作称为“盒子变形”（Box morphing），可用于顺应具有三维几何分量的曲面。


要在曲面上放置扭曲的盒子，必须将曲面域划分为一个网格。通过将盒子的每个端点向对应的目标点生成向量，并移动到相应的位置。

1.5.1.3 Box Morph盒子变形实例

在此示例中，我们将使用Box Morph将一个物体沿Nurbs曲面流动。


  1. 完成后的效果
  2. 原始Box的物体
  3. 曲面网格划分
  4. 扭曲的盒子排列在曲面上

案例步骤：

  1. 打开一个犀牛文件
  2. Params / Geometry / Surface –将Surface电池拖放到画布上
  3. Params/Geometry/Geometry - 将Geometry电池拖到画布上
  4. 右键单击Surface电池，然后选择“Set one Surface” –在Rhino视口中选择目标曲面
  5. 右键单击Geometry电池，然后选择“Set one Geometry” –选择您的Rhino几何物体
  6. Maths / Domain / Divide Domain2 –将Divide Domain2电池拖放到画布上
  7. Params / Input / Silder – 将三个Slider拖到画布上
  8. 将Slider设置以下内容：整数；Min：0；Max：10；Value：5
  9. 将其他两个Slider进行同样的设置
  10. 将Surface电**池**的输出连接到Divide Domain2组件的Domain（I）输入端
  11. 将第一个Slider连接到Divide Domain2组件的U Count（U）输入端
  12. 将第二个Slider连接到Divide Domain2组件的V Count（V）输入端
  13. Transform / Morph / Box Surface – 将“Surface Box”组件拖动到画布上
  14. 将Surface电池的输出连接到Surface Box电池的Surface（S）输入端
  15. 将Divide Domain2电池的Segements（S）输出连接到Surface Box电池的Domain（D）输入端


您应该看到一个扭曲的盒子组合，顺应着参考曲面流动。更改“ U”和“ V”的Slider以更改数量，并使用高度slider调整其高度。

  1. 将第三个Slider连接到Surface Box组件的高度（H）输入端
  2. Surface / Primitive/ Bounding Box – 将Bounding Box组件拖到画布上
  3. Transform / Morph / Box Morph – 将“Box Morph”电池拖放到画布上
  4. 将Geometry电池的输出连接到Bounding Box组件的Content（C）输入端
  5. 将Geometry电池的输出连接到Box Morph组件的Geometry（G）输入端
  6. 将Bounding Box电池的Box（B）输出连接到Box Morph电池的Reference（R）输入端
  7. 将Surface Box电池的Twisted Box（B）输出连接到Box Morph电池的Target（T）输入端


现在，可以看到原始体块顺应曲面曲面进行了变形流动。
